## Game: ``Snake``
Simple snake game.
Eat apples to earn points. Each apple eaten increases the length of the snake.

## Features
* Several maps
* Supports screen scrolling for bigger maps
* Maps are tile based
* Highscore table
* Eating apples extends snake size
